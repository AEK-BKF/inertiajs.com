# File uploads

The trick to uploadingÂ files via `xhr` is using the `FormData` object. You can learn more about `FormData` interface [here](https://developer.mozilla.org/en-US/docs/Web/API/FormData). Here is a simple example of using `FormData` with Inertia.

```js
var data = new FormData()
data.append('first_name', first_name || '')
data.append('last_name', last_name || '')
data.append('email', email || '')
data.append('password', password || '')
data.append('photo', photo || '')

Inertia.post('/users', data)
```
